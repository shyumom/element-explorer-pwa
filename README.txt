원소탐험대 PWA 실행 방법

이번 버전 추가 사항
- 118개 원소 전체 데이터 포함
- 원소별 시각 카드 이미지 118개 포함(element-images 폴더)
- 정답, 오답, 학습완료, 카드 매칭, 배지 획득 효과음 포함
- 별도 mp3 파일 없이 브라우저 Web Audio API로 효과음 생성
- PWA 설치형 파일 포함(manifest.webmanifest, sw.js, icons)

실행 방법
1. 압축을 풉니다.
2. VS Code에서 원소탐험대_PWA 폴더를 엽니다.
3. index.html을 Live Server로 실행합니다.
4. http://localhost 또는 http://127.0.0.1 주소에서 실행해야 PWA와 Service Worker가 작동합니다.
5. 크롬에서는 주소창 설치 아이콘 또는 앱 내 '앱 설치' 버튼을 이용합니다.
6. 아이패드에서는 GitHub Pages/Netlify/Vercel 등에 올린 뒤 Safari에서 '홈 화면에 추가'를 선택합니다.

주의
- file://로 직접 열면 PWA 설치 기능과 Service Worker가 작동하지 않습니다.
- 효과음은 사용자가 화면을 한 번 클릭한 뒤부터 정상 재생됩니다. 이는 브라우저의 자동재생 제한 때문입니다.
- 학습 기록은 localStorage에 저장됩니다.


수정 이력 - 퀴즈/아이패드 UI 보정
- 기호 맞히기 퀴즈의 보기에서 원소기호를 제거했습니다.
- 보기에는 한글 원소명만 표시됩니다.
- 정답/오답 피드백에서만 원소기호를 함께 표시합니다.
- iPad Air 4 가로 화면 기준으로 주기율표 타일, 우측 원소 패널, 이미지 카드, 설명 영역 높이를 조정했습니다.
- PC 넓은 화면에서는 1180 x 820 기준으로 앱 영역이 고정되어 화면 넘침을 줄였습니다.


수정 이력 - 놀이 방식 변경
- 기존 카드 짝 맞추기 놀이를 제거했습니다.
- 새 놀이 1: 빈칸 채우기
  · 원소 키워드를 제시하고 원소기호, 원자번호, 원소명, 상태 중 하나를 직접 입력합니다.
- 새 놀이 2: 원소 이름 맞히기
  · 원자번호, 원소기호, 분류, 상태, 설명을 보고 한글 원소명을 직접 입력합니다.
- 정답 시 +40XP, 3연속 정답 시 추가 +30XP 보너스를 지급합니다.
- 오답 시 힌트를 제시하고 재도전할 수 있습니다.


수정 이력 - 놀이 힌트 흐름 보완
- 놀이 화면의 '오늘의 원소 키워드'를 삭제했습니다.
- 빈칸 채우기와 원소 이름 맞히기에서 정답이 먼저 노출되지 않도록 수정했습니다.
- '잘 모르겠어요. 학습 탭에서 다시 보고 올래요' 버튼을 추가했습니다.
- 해당 버튼을 누르면 문제의 원소가 학습 탭에서 자동 선택됩니다.
- 학습 탭을 보고 다시 놀이 탭으로 돌아와도 기존 문제는 유지됩니다.


수정 이력 - 놀이 유형 재정리
- '빈칸 채우기'를 제거했습니다.
- 놀이 유형을 2가지로 명확히 정리했습니다.
  1. 원소기호 맞추기: 한글 원소명, 원자번호, 설명을 보고 원소기호 입력
  2. 원소 이름 맞히기: 원소기호, 원자번호, 설명을 보고 한글 원소명 입력
- 원소기호 맞추기에서는 보기 또는 키워드에서 원소기호를 노출하지 않습니다.
- 학습 탭으로 힌트를 보러 갔다가 돌아와도 문제는 유지됩니다.


수정 이력 - 짧은 음성 안내
- 학습 탭에 '이름·기호 듣기' 버튼을 추가했습니다.
- 놀이 정답 후 '정답 듣기' 버튼이 표시됩니다.
- 퀴즈 정답 후에도 '정답 듣기' 버튼이 표시됩니다.
- 음성은 자동 재생하지 않고, 버튼을 눌렀을 때만 재생됩니다.
- 음성 내용은 길게 설명하지 않고 '[원소명]. 원소기호는 [기호 읽기].'만 읽습니다.
- 예: 수소. 원소기호는 에이치.
- 예: 정답, 산소. 원소기호는 오.
- 효과음과 음성 안내는 별도로 제어됩니다.
